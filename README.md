# Cloud Security Copilot

Cloud Security Copilot is an AWS-based security platform that identifies over-permissioned IAM roles, analyzes CloudTrail activity, calculates risk scores, and recommends least-privilege policies.

## Features

- IAM Role Discovery
- Policy Risk Analysis
- CloudTrail Usage Analysis
- Risk Scoring Engine
- AI-Powered Recommendations
- Security Dashboard
